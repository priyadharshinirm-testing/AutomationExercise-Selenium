# Selenium Automation Project

This project automates the "Register User" functionality in AutomationExercise website.

Tools Used:
- Java
- Selenium WebDriver
- Maven
- Eclipse

Test Scenario:
Register a new user in the AutomationExercise website.

Steps Automated:
1. Launch browser
2. Navigate to http://automationexercise.com
3. Click Signup/Login
4. Enter Name and Email
5. Fill account details
6. Create account
7. Verify account created
8. Delete account
